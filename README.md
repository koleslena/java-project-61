# Проект Игры разума

Игры разума — это набор из пяти консольных игр. Каждая игра представляет собой математические вопросы, на которые нужно дать правильные ответы. После трёх правильных ответов игра считается пройденной.

## Игры 

- Чётное число — определение чётного числа.
- Калькулятор — арифметические выражения, которые необходимо вычислить.
- НОД — определение наибольшего общего делителя.
- Прогрессия — поиск пропущенных чисел в последовательности чисел.
- Простое число — определение простого числа.

## Запуск

<code>./build/install/app/bin/app </code>
